# 2. 问题解决

注意日志中的这句话

 Could not write JSON: Null key for a Map not allowed in JSON (use a converting NullKeySerializer?)
1

说明在我们需要转换的Map中包含一个元素，且这个元素的Key为 null

经过逐一排查后，将map的key设置为 null 这部分代码修复，就好了。
